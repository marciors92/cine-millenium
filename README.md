Aplicação web desenvolvida para gerenciar as sessões comemorativas da trilogia "O Senhor dos Anéis". O projeto foca em **Qualidade de Software** e **Automação de Deploy**.

🚀 **[Acesse o Cine Millenium aqui](https://marciors92.github.io/cine-millenium/)**

---

## 🛠️ Tecnologias e Ferramentas

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Linguagem:** [TypeScript](https://www.typescriptlang.org/) (Tipagem Estática)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
* **Testes:** [Jest](https://jestjs.io/) & React Testing Library
* **DevOps:** GitHub Actions (Pipeline CI/CD)

---

## 🏗️ Estrutura do Projeto

* `/app`: Interface principal e lógica de rotas.
* `/__tests__`: Testes de unidade para garantir a integridade dos dados.
* `/.github/workflows`: Automação de testes e deploy para GitHub Pages.
